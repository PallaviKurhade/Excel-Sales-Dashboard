# Excel-Sales-Dashboard

📊 Sales Dashboard Project (Microsoft Excel)

I developed an interactive Sales Dashboard using Microsoft Excel to analyze and visualize business performance data. The project focuses on identifying sales trends, profit performance, and key business insights to support data-driven decision-making.

# Dataset used
<a href="https://github.com/PallaviKurhade/Excel-Sales-Dashboard/edit/main/README.md">Dataset</a>

🔹 Key Responsibilities:
Cleaned and organized raw sales data using Excel functions
Used Pivot Tables and Pivot Charts for dynamic data analysis
Created KPIs such as Total Sales, Total Profit, and Quantity Sold
Analyzed sales by category, region, and time period
Designed an interactive dashboard with slicers for better user experience

🔹 Tools & Skills Used:
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Data Analysis
Dashboard Design
Basic Data Visualization

🔹 Outcome:
This project enhanced my analytical thinking, Excel dashboard skills, and ability to convert raw data into meaningful business insights.
